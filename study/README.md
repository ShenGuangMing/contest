# 异或运算
异或运算是在底层的二进制数上进行运算得出的

## 异或运算的特点
- 相同为0
- 不同为1
- 无进位相加

例如 
```text
异或运算        
  a:00010110                    
  b:00011011           
a^b:00001101   
  
无进位相加
 a:00010110                    
 b:00011011           
a+b:00001101(每个都没进位)
```
- 异或运算同时也满足**交换律**和**结合律**
```text
a^b = b^a 
(a^b)^c = a^(b^c)
``` 
## 使用异或交换
假如有a = 3，b=7两个数
```text
a = a^b; //a等于了3^7，b等于7
b = a^b; //带入，b=3^7^7化简：b=3，a=3^7
a = a^b; //带入，a=3^7^3化简：a=7，b=3
```
